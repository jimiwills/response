NAME

Tk::MIMEApp - The great new Tk::MIMEApp!

VERSION

Version 0.03

SYNOPSIS

ISA Tk::Notebook. Can load MIME Multipart file with application/x-ptk.markdown parts, converting them to Tk (with Tk::MarkdownTk) for document-driven applications.

SUBROUTINES/METHODS

loadMultipart

Method that takes a filehandle and adds a page to the notebook for each application/x-ptk.markdown part. Will also add menu items to the toplevel menu for application/x-yaml.menu parts.

yaml2menu

Accepts some yaml (scalar) and a Tk::Menu as arguments, and populates the menu from the yaml. Uses YAML::Perl for parsing.

array2menuitems

Takes a Tk::Menu and and arrayref describing items and implements them. Called by yaml2menu.
